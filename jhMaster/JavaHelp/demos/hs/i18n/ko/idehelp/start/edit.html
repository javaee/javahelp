자습서 2: 프로젝트 속성 수정하기

이 과에서는, 프로젝트 속성이란 무엇인지를 배우고 실행 속성과 빌드 속성을 수정하는 방법을 익힐 것입니다. 본 자습서는 세 개의 과로 나누어져 있습니다. 자습서에서 최대한 유익을 얻으려면, 먼저 프로젝트 속성에 대한 해설을 검토한 다음에 그 다음 두 과를 익혀야 합니다.
  • 중요한 개념: 프로젝트 속성에서는 프로젝트 속성이란 무엇인지를 설명하고 속성을 Java Development Environment에서 사용하는 방식을 설명합니다. 제 1과로 넘어가기 전에 이 개념들을 이해해야 합니다.
  • 제 1과: 실행 속성 수정하기에서는 애플릿을 실행하는 방식을 지정하는 방법을 보여줍니다.
  • 제 2과: 빌드 속성 수정하기에서는 빌드 기법 차이를 지정하는 방식을 보여줍니다.
